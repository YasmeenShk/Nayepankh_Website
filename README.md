# NayePankh Foundation — Website

A single-page website built for **NayePankh Foundation**, an NGO working to uplift underprivileged communities through food, education, and care. This project was created as part of an internship assignment.

🔗 **Live Demo:** [yasmeenshk.github.io/Nayepankh_Website](https://yasmeenshk.github.io/Nayepankh_Website/)

---

## 📌 About

NayePankh Foundation is a student-led NGO providing free food, education, hygiene kits, and relief support to underprivileged communities across India. This website was designed to present their mission, work, and a way for visitors to contribute — all in a clean, modern, single-page layout.

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Sticky navigation bar
- Mobile hamburger menu
- Hero, About, Donate, and Contact sections
- Dark monochromatic **"Ink Wash"** color theme
- Built with utility-first CSS for fast, maintainable styling

## 🛠️ Tech Stack

- **HTML5** — page structure
- **Tailwind CSS** — styling
- **JavaScript** — mobile menu interactivity

## 🚀 Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/yasmeenshk/Nayepankh_Website.git
   ```
2. Open `index.html` in your browser — that's it, no build step required to view it.
